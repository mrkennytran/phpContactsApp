### Database Schema
- ##### **login system db**
	- USERS table
		- user_id
		- username
		- email
		- password

- #### contacts db
	- ADDRESS TABLE
		- street
		- city
		- state
		
	- PEOPLE TABLE
		- name
		- company
		- notes
		- favorite
		- active
		- address_id

	- PHONE NUMBER TABLE
		- phone_id
		- person_id
		- phone_number
		
	- EMAIL TABLE
		- email_id
		- person_id
		- email

### CRUD
- #### CREATE
	- Create new contact page
	
- #### READ
	- Display all contacts
	
- #### UPDATE
	- Update existing contact
	
- #### DELETE
	- Set contact as inactive
	- Then Delete inactive contacts
